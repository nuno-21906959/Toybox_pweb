    Técnicas usadas

    Caracteristicas Elementos/Técnicas Exemplos
    Etiquetas Foram usadas as etiquetas <head>,<body>,<div>,<nav>,<section>, <aside>,<footer>,<ul>,<li>,<h1>,<input>,<p>,<a>, <svg>,<img>,

    Foi usado <div>, por exemplo para fazer o wrapper para as paginas;

    Foi usado <nav>, <ul> e <li>para o menu no header;

    Foi usado <section>, por exemplo para dividir as secçoes na pag. inicial;

    Foi usado <aside> para fazer a sticky note para deixar feedback;

    Foi usado <input>, por exemplo, no footer, para a inscrição na newsletter;

    Foi usado <svg> e <img>, por exemplo, na wordcloud desta página.
    Layouts Foi usado CSS Grid e Flexbox para fazer o layout do website.

    Foi usado CSS Grid, por exemplo, na parte sobre os eventos da pag. princípal;

    Foi usado Flexbox, por exemplo, na parte das marcas na pag. princípal.
    Responsividade Foram usadas media queries que adaptam o layout para telemovel e PC consoante o tamanho da janela do browser. Foram usadas medias querias para as seguintes widths: 1024px, 768px, 540px.
    Seletores Foram usados seletores de tipo, classe, id, atributo, descentes e filhos

    Foram usados seletores de tipo para estilizar, por exemplo, o body, os hyperlinks, as imagens e o header;

    Foram usados seletores de classe, por exemplo, para estilizar a barra mais escura em baixo de header;

    Foram usados seletores de id, por exemplo, para meter os backgrounds nos containers com o titúlo das páginas "Sobre a Empresa", "Eventos", etc.

    Foram usados seletores descendentes, por exemplo, para estilizar o texto e as àreas dentro do header;

    Foram usados seletores filhos, por exemplo, para estilizar o input na pagina de feedback.
    Animações Foram usados keyframes para fazer as animações. Na página principal há 2 animações, uma de mudança de cor e tamanho, e uma de translação nas imagens das marcas.
